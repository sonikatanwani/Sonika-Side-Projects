This project was created using WebStorm IDE
Following Library were included:-
      AngularJS
      JQuery.js
      Validator.js
      BootStrap.css ( For Responsive)
      Font-Awesome ( For icons)
      NodeJS/Karma/Jasmine ( For Unit Test )
This application is Single Page with Start Point as geolocation.html
To Run Test Case, run the batch file at following location Test\StartUnitTest.bat
To view Test output, browse following file Test\output\units.html
Controller have be designed to support Unit Test.
